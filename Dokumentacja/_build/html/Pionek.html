Pionek module

class Pionek.Pionek(kolor)

Klasy bazowe: object

Klasa reprezentująca pionka do gry.

Atrybuty klasy:

-pozycja: parametr typu NoneType, Pole lub str(„zdjęty”, „banda”), który mówi, gdzie aktualnie znajduje się pionek

-kolor: parametr typu str, który mówi, jakiego koloru jest pionek do gry (czarny lub czerwony)

gdzie()

Zwraca aktualną pozycję pionka.

Zwraca:

Aktualna pozycja pionka (pole, None,”zdjęty” lub „banda”).

Typ zwracany:

Pole, NoneType lub str

podaj_kolor()

Zwraca kolor pionka.

Zwraca:

Kolor pionka („czarny” lub „czerwony”).

Typ zwracany:

str

zmien_pozycje(nowa_poz)

Zmienia pozycję pionka na podane pole.

Parametry:

nowa_poz (Pole lub str) – Nowa pozycja pionka (pole,”zdjęty” lub „banda”).

Zwraca:

None

Typ zwracany:

NoneType
